# ExampleMetric
## Usage
### Step 1: Install Node.JS
Download from www.nodejs.org
### Step 2: Install Eslint
```
$ npm install eslint
```
### Step 3: Install sloc
```
$ npm install sloc
```
### Step 4: Run nodejs
```
$ node app.js
```
